# Upload instructions

1. Unzip `Heat-Perception-2_upload_materials.zip` on your computer.
2. Open the GitHub repository: https://github.com/MeiziYou-Research/Heat-Perception-2
3. Click `Add file` -> `Upload files`.
4. Drag the folders and files from `Heat-Perception-2_upload_materials/` into the GitHub upload area.
5. Use this commit message:

   Add codebook, prompt variants, validation examples, and supplementary audit materials

6. Click `Commit changes`.

If GitHub does not accept a whole folder drag-and-drop, upload the folders one by one: `codebook/`, `prompts/`, `validation_examples/`, and `supplementary/`, then upload `README.md`, `DATA_AVAILABILITY.md` and `CODE_AVAILABILITY.md` at the root level.

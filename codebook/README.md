# Codebook

This folder contains the codebook materials used for closed-set LLM annotation of urban heat perception.

## Files

- `codebook_full_attention_emotion.txt`: complete plain-text coding manual.
- `Supplementary_Table_1_attention_taxonomy.csv`: structured attention taxonomy with 24 subcategories.
- `Supplementary_Table_2_emotion_scheme.csv`: structured emotion scheme with seven labels.

The plain-text codebook consolidates the annotator role, task scope, in-scope and out-of-scope criteria, single-label decision rules, attention definitions, emotion definitions, output schema and quality-control checklist.

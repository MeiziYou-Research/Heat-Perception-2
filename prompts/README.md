# Prompt variants

This folder contains the prompt variants used for prompt-ablation analyses.

The production prompt for both attention and emotion classification was V2.

Prompt variant definitions:

- V0: full codebook definitions and decision rules.
- V1: label names only.
- V2: full codebook definitions plus annotated examples; used as the production prompt.
- V3: minimal instruction.
- V4: domain-expert role frame.

Attention prompts are stored in `attention/` and emotion prompts are stored in `emotion/`.
